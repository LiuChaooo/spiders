Nike React Hyperdunk 2017 Flyknit伴随德雷蒙德•格林（Draymond Green）赢得总冠军，搭载了耐克专有的全新Nike React缓震平台，带来了舒适性，支撑性以及响应性的能量反馈。
自2008年面世至今，Hyperdunk系列已经成为耐克篮球鞋设计的缩影。每一代Hyperdunk的造型都同时彰显着全新的创新，是对耐克篮球鞋理念最纯粹的呈现。Nike React Hyperdunk 2017 Flyknit再续传奇，以明快、极简的审美观念，将现代融入经典之中，也将耐克篮球的理念推向新的高度。
首先是鞋底，Nike React Hyperdunk 2017 Flyknit的外底抓地力纹理设计以数据分析为基础，结合耐克运动研究实验室提供的压力分布图，提供稳定的支撑性，配合准确分布的圆形节点，在需要的地方提供抓地力。密封在球鞋内部，Nike React核心同时带来三重优势：轻量性、响应性和耐久性。位于Nike React缓震和外底之间嵌有一块承托盘，为球鞋加强硬度，控制扭转。在篮球运动中，这些优势能够帮助运动员实现快速突破、顺畅起跳并使表现更为持久。格林表示： “这款球鞋既轻量，又具有回弹性，可以让我在球场上随心所欲的冲刺和移动。”
Flyknit鞋面，为球鞋提供了定点的延展性和透气性，鞋口则是熟悉的80年代风格，用当代材料演绎而成。与柔软有弹性的Nike React泡棉类似，Flyknit鞋面平衡演绎了两全其美的审美，既能立刻让人联想到过去的经典，同时又以先锋设计致敬未来。值得注意的是，精确设计的鞋领，能够提供自然的灵活性和定制般的贴合。
Nike React Hyperdunk 2017 Flyknit将于8月3日正式发布，拥有的冠军血统，让一切变得圆满。
“能够首先穿上最新款的Hyperdunk，赢得总冠军，对我来说非比寻常，” 格林感叹道。
